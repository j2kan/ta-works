Step Process Description
1 Change Password If you are a new AC, the password needs to be changed to maintain a secure system. In addition, use your discretion to determine when the password should be changed.
2 Upload Courses For each term, a csv file needs to be uploaded that identifies important information regarding each of the courses that will be offered. This task must be completed prior to the TA Application Form becoming open.
3 Open/Close the TA Application When the upcoming terms courses have been added to the system, the AC must make the TA Application Form 'open'. This allows graduate students to apply for a TA position. The AC must then close the form when the application period has ended. The AC may also preview the application form here.
4 Change Application Form The TA Application Form has a front matter section. This section identifies important information that graduate students need to be aware of prior to applying. Should this information change, you are able to change the content through this process.
5 Review Applicants The AC is responsible for ensuring all applicants are eligible to hold a TA position. The AC can export applicant information to send to the GC for verification. The AC can also adjust the ratings students submitted for courses if needed.
6 Assign Number of TAs The AC is responsible for assigning the number of TA's that are required for each course in the upcoming term.
7 Send Ranking Links, Monitor Replies Once the application period has ended, the AC is responsible for notifying Instructors that they need to rank appplicants. System generated emails can be sent out to all Instructors here. This step will also provide a summary of which Instructors have ranked their applicants. The AC also has the ability to override Instructor preference for each course in this step.
8 Export Results The AC can export a complete list of (1) the number of TAs required for each course and (2) student and instructor rankings by course. This is for the AC's historical records.
9 Assign Teaching Assistants The AC has to assign teaching assistants to courses. This step allows the AC to run an assignment aglorithm and export the results.
